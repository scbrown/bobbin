# CLI Overview

> **Status: Draft**

Bobbin provides 16 commands for code analysis, search, and context assembly.

## Global Flags

| Flag | Description |
|------|-------------|
| `--json` | Output in JSON format |
| `--quiet` | Suppress non-essential output |
| `--verbose` | Show detailed progress |
| `--server <URL>` | Use remote bobbin server |
